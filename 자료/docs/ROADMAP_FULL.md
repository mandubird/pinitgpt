## 📅 Phase 1: MVP 개발 및 UX 설계 (D-Day ~ D-3)

### 1.1 핵심 기능 분기 (Free vs Pro)

| 구분 | 무료 (Free) | Pro / Early Supporter |
|------|-------------|------------------------|
| **Pin 저장** | 프로젝트당 최대 5개 | 무제한 |
| **프로젝트** | 1개 (현재 채팅방) | 무제한 (모든 채팅방 관리) |
| **특수 기능** | 유형 분류 ✔ | ⭐ 중요 표시, 보기(전체/이어서), 검색, 정렬, 아카이브 ✔ |

| **데이터** | 로컬 저장 (localStorage) | 로컬 저장 (추후 클라우드 동기화) |

### 1.2 Early Supporter 전용 UX (자부심 & 신뢰)

- **🌱 전용 뱃지**: 사이드바 하단 및 설정에 Early Supporter 뱃지 노출

## 🛠 Phase 3: 수익 모델 고도화 (월정액 전환)

**LTD 1,000명 달성 후** 플랜 세분화.

### 3.1 플랜별 가격 및 기능 비교

| 기능 | Free (무료) | Pro ($4.99/mo) | Pro+ ($9.99/mo) |
|------|-------------|----------------|------------------|
| **기본 핀/프로젝트** | 제한적 (5개/1개) | 무제한 | 무제한 |
| **클라우드 동기화** | ❌ | ✔ (개인) | ✔ (멀티 디바이스) |
| **AI 스마트 정리** | ❌ | ❌ | ✔ (자동 요약/분류) |
| **협업 (Workspace)** | ❌ | ❌ | ✔ (팀 공유/동시 편집) |
| **Export 강화** | 기본 (Copy) | MD / PDF 추출 | Notion/Slack 연동 |

### 3.2 플랜별 핵심 상세 기능

#### 1) Pro ($4.99/month) – 개인 생산성 중심

- **인증 및 클라우드**: 개인 계정 기반 로그인, 모든 기기 데이터 동기화
- **관리 기능**: 무제한 프로젝트, 고급 필터링·검색
- **Export**: Pin을 Markdown / PDF로 내보내기

#### 2) Pro+ ($9.99/month) – AI 자동화 및 협업 중심

**💎 AI 기능**

- 자동 요약: 긴 답변 핀 시 AI가 핵심 3줄 요약 자동 생성
- 스마트 분류: Pin 내용 분석 → 프로젝트·태그 AI 자동 추천

**💎 협업 (Collaboration)**

- Shared Workspace: 팀원과 공유 프로젝트 보드
- Live Link: 프로젝트 핀 리스트를 외부 URL로 실시간 공유

**💎 연동 강화**

- Zapier / Webhook: Pin 시 Notion 자동 전송, Slack 알림

**💎 다국어 확장 (Pro+ 개발 시점)**

- **일본어 (ja)**: `locales/ja.json` 추가, 사이드바 언어 선택에「日本語」추가. → docs/I18N.md §6
- **독일어 (de)**: `locales/de.json` 추가, 사이드바 언어 선택에 "Deutsch" 추가.

## 🏗 Phase 4: 기술 인프라 (인증 & 결제)

### 4.1 시스템 구성

| 구성요소 | 기술 |
|----------|------|
| **인증 서버** | Node.js + Express. 구독 만료일·플랜 등급(Tier) 관리 |
| **Database** | Firebase Firestore (실시간 동기화) + Redis (캐싱) |
| **결제** | Stripe (구독 자동 갱신, 플랜 업/다운그레이드) |

### 4.2 보안 및 데이터 원칙

- **Privacy First**: 대화 원본은 Pin된 데이터 외 수집하지 않음
- **Security**: AI 분석 시 데이터 암호화, 학습용 미사용

# Cursor에서 코드/폴더가 안 보일 때

- **오른쪽에 "New Agent"만 보이고 코드·폴더가 안 보이는 경우**
  - **왼쪽에 폴더 트리(Explorer) 보이게**: 메뉴 **View → Explorer** 또는 단축키 **`Cmd+Shift+E`** (Mac) / **`Ctrl+Shift+E`** (Windows).
  - **에디터에 코드 보이게**: 왼쪽 Explorer에서 파일을 클릭하면 가운데(또는 오른쪽)에 코드가 열립니다.
  - **오른쪽을 채팅이 아닌 코드로 쓰고 싶을 때**: 채팅/Agent 패널을 닫거나, 에디터 탭을 오른쪽으로 드래그해서 **분할 편집**하면 왼쪽·오른쪽에 각각 코드를 띄울 수 있습니다.

- **요약**: 폴더/파일 목록 = **왼쪽 Explorer** (Cmd+Shift+E). 코드 = **Explorer에서 파일 클릭**.

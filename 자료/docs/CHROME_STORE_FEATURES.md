# PinitGPT — 무료 / Pro 기능 정의서 (Chrome 웹스토어용)

ChatGPT 대화에서 중요한 답변을 Pin으로 저장·분류하고, 채팅(프로젝트) 단위로 관리하는 Chrome 확장 프로그램.  
아래는 **현재 구현된** 무료(Free) 버전과 Pro 버전의 기능 정의입니다. 크롬 웹스토어 상세 설명·스크린샷 구성 시 참고용.

---

## 한 줄 요약 (스토어 짧은 설명용)

**Free**: ChatGPT 대화에서 중요한 답변을 Pin으로 저장·분류 (채팅 1개, Pin 5개).  
**Pro**: 모든 채팅 Pin 통합 보기, 검색·정렬·태그·내보내기(CSV/Notion), 무제한 Pin.

## 2. 무료(Free) 버전 상세

### 2.1 제공 기능

- **Pin 저장**: ChatGPT 메시지 옆 📌 버튼으로 Pin 저장. 유형 선택: 🔵 In progress, 🔴 Needs resolution, 🟢 Review, 🟡 Reference, 📌 Continue.
- **사이드바**: 현재 채팅의 Pin만 목록 표시. 정렬(최신순/오래된순), 카테고리(All / Important / Needs resolution / In progress / Review / Reference / Continue) 필터.
- **Pin 클릭**: 해당 메시지로 스크롤 이동 및 하이라이트.
- **저장 제한**: **채팅 1개**, **채팅당 Pin 5개**. 초과 시 안내 문구 + Pro 업그레이드 링크.
- **다국어**: English / 한국어. 사이드바 하단 🌐에서 전환 (새로고침 없이 반영).
- **데이터**: 로컬 저장(localStorage). 외부 서버 전송 없음.

### 2.2 Pro 잠금 표시 (무료 사용자에게 보이는 것)

- 보기 버튼 **[전체]**, **[📌 이어서]** — 흐리게 + 🔒, 클릭 시 "Pro에서 사용 가능" 토스트.
- 검색창 — "🔒 Search (Pro)", 입력 비활성.
- 내보내기 버튼 — 클릭 시 Pro 업그레이드 안내.
- ⭐ 중요 표시, 태그 무제한 — Pro에서 사용 가능 안내.

## 3. Pro 버전 상세

### 3.1 무료 대비 추가·확장 기능

- **보기 (View)**  
  - **전체**: 저장한 모든 채팅의 Pin을 한 목록에서 조회.  
  - **현재 채팅**: 무료와 동일.  
  - **📌 이어서**: "이어서 볼 Pin"만 필터.
- **검색**: 제목·내용 기준 실시간 검색. 검색어 노란색 강조.
- **정렬**: 최신순 / 오래된순 (무료와 동일, 검색과 함께 사용 가능).
- **카테고리**: 카테고리별 이름 편집(✏️). Needs resolution, In progress 등 사용자 정의 라벨.
- **⭐ 중요(Star)**: Pin별로 중요 표시. Important 카테고리에서 일괄 조회.
- **태그**: Pin당 무제한 태그. 태그 클릭 시 해당 태그로 필터.
- **내보내기**:  
  - **CSV**: label, chatTitle, text, tags, createdAt. 현재 채팅 Pin 다운로드.  
  - **Notion**: Markdown 테이블 형식. 동일 컬럼.
- **Pin·채팅 제한**: 무제한.
- **Early Supporter**: 라이선스 활성화 시 사이드바 하단에 🌱 Early Supporter 뱃지. "이 라이선스는 만료되지 않습니다" 툴팁.

### 3.2 활성화 방법

- **라이선스 키**: 확장 아이콘 클릭(popup) 또는 사이드바 하단에서 Gumroad 구매 후 받은 라이선스 키 입력 → "💎 PRO activated" 후 Pro 기능 전체 해제.

## 4. 공통 사항 (Free / Pro 동일)

- **지원 환경**: Chrome(Chromium). ChatGPT(chatgpt.com) 페이지에서 동작.
- **저장 위치**: 브라우저 로컬(localStorage). 계정·서버 동기화 없음.
- **언어**: English, 한국어. 사이드바 하단 🌐으로 전환.
- **개인정보**: Pin 데이터는 사용자 기기 내에만 저장. 기본 동작으로 외부 전송 없음.
